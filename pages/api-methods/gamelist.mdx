# getGameList Method

The `getGameList` method returns a list of all games available for the selected currency.

When opening a game session using the [getGame method](/api-methods/get-game), you should **always use the `id_hash` value** provided in the game list.

Games that support free spins via the [addFreeRounds method](/api-methods/add-freerounds) are marked with `freerounds_supported: true`.

---

## Method Request

```json
{
  "api_login": "d13b1ceb-6209-4163-b1cf-f304c28b81ec",
  "api_password": "dqRin2nfLosij2n88",
  "method": "getGameList",
  "show_additional": true,
  "show_systems": 0,
  "list_type": 1,
  "currency": "USD"
}
```

---
